Website Pre-Order

Features:

  • Admin can create multiple Pre-order Configuration settings, but only one can be active at a time.
  • User can place pre-order for only one partial pre-order product.
  • Admin can set a custom message to be displayed on Pre-order product.
  • Admin can set percentage of the amount that is to be paid in case of partial pre-order product.
  • Using this module registered customer can pay full or partial payment that has been set by the admin in the configuration for the Pre-order Products or in the Global configurations.
  • Admin can set the mode of notification either to 'Automatic' or 'Manual' according to the availability of the pre-order product.
  • Using this module Unsigned customer can place a pre-order but they have to pay the full amount at the time of the order.

Custom Message and Product Available Date

Pre-order Product with Custom Message and Product Available Date

About Custom Message:

  • If you want to display any customized message below the price details of the pre-order product then in "Website Pre-order Configuration" write the customized message.

About Product Available Date:

  • In the Configuration settings, if you want to make product "Available date" visible then 'enable' the 'Available Date Visible on Pre-order Product' option.
  • In the Configuration settings, if you want to make product "Available date" invisible then 'disable' the 'Available Date Visible on Pre-order Product' option.

Customize Label for Pre-order Button

About Button Label Customization:

  • To change the label of the pre-order button, in 'Website Pre-order Configuration' write the desired name in the field named 'Add To Cart Button Text'.

Pre-Order Quantity Range

Pre-order Product with Pre-order Quantity Range

About Pre-order Quantity Range:

  • If you want to display Pre-order Quantity Range then in "Website Pre-order Configuration" enable Display Pre-ordered Quantity Range option.
  • Set the Minimum and Maximum Pre-ordered Quantity in configuration.

Pop-Up: Can't Add Other product

About Pop-up:

  • If a partial pre-order product is already present in your cart, then you can't add any another product.
  • If a regular product or complete pre-order product is already present in your cart, then you cannot add any partial pre-order product in your cart and vice versa.
