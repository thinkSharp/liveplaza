Odoo SMS Notification

Odoo SMS Notifications module is the base module that allows you to send SMS on mobile number from Odoo by integrating SMS Gateways with Odoo.

You Would need to install an SMS Gateway with this module to Send message to people. Scroll Down to see the list of available SMS Gateways for Odoo.

Connect with your customers with Mobile SMS Notifications

The module installs the essential features in your Odoo do you can send SMS notifications to your Customers directly from Odoo.

Send mobile messages to single customer or in bulk to many customers and vendors with a click.

Save Odoo SMS templates to use them in the SMS you send. 

Send Order Updates and Notifications texts on their Mobile Phones

Increase customer satisfaction and keep them notified about their orders, updates etc with mobile messages.

Create Odoo SMS templates for different Conditions. These SMS are automatically sent when the assigned condition(s) is(are) fulfilled in Odoo.

Send Discount, Deals, and Important Updates in your Store right to the phones

Improve Customer Engagement and personalization by reaching them via mobile phone.

Create Odoo SMS template for Upcoming Sales, season discount, customers’ birthdays, etc. and send them to one or more people using Odoo SMS Notification Module.

Integrate Multiple SMS Gateways into Same Odoo

Simple Integration steps facilitates quick integration of various SMS Gateways with Odoo.

You can set the priority of the integrated SMS Gateways to choose which one to use while sending the SMS.

Odoo SMS Notification is a base module for Sending SMS in Odoo. It is designed to provide the structure and flow to enable the SMS feature in Odoo.

In addition to Odoo SMS Notification module, you would also need to integrate an SMS gateway with Odoo to send SMS notifications to your customers.

Detailed Features List

Here is the detailed list of features for Odoo SMS Notification Module.

Provides Structure to Odoo SMS module

  • The module install all the necessary functions to send SMS from Odoo.
  • Send mobile message notifications and updates to customers, vendors or both.

Integrate Multiple SMS Gateways

  • Integrate various SMS Gateways with Odoo such as Skebby, Plivio, etc.
  • Set Gateway priority so Odoo automatically picks desired Gateway to send SMS.

Create Odoo SMS Groups

  • Create SMS groups to send the message in bulk to people.
  • Single group or multiple groups can be selected to send the Odoo SMS.

Create Odoo SMS Templates in Advance

  • Create and Save SMS templates to use later while sending the SMS.

Enable/disable SMS Feature on User Basis

  • You can give access to Odoo SMS feature at user level.
  • Only user with Odoo SMS feature enabled can send messages to people in Odoo.

Send Single or bulk Sms to customers / send SMS individually or in bulk

  • Send message to single customer/vendor using the SMS form.
  • Or choose entire groups or multiple people to send the Odoo SMS.

Set SMS Templates For Global or on Conditions-based Use

  • The templates can be set for use at - Global level or Condition-based.
  • Global Odoo SMS templates are available for all the Odoo SMS.
  • Condition-based are triggered for Odoo SMS only when the set condition is met.

Auto send SMS to people

  • Assign conditions to automatically send SMS to customers from Odoo.
  • Odoo automatically picks the set SMS template once the condition is met in Odoo.
  • The SMS is then automatically sent to the specific customer(s).

Creating Conditions Based SMS Templates

  • Choose from different given conditions to set for the Odoo SMS template.
  • Set conditions such as; order confirmed, invoice validated, order placed, etc.

Manage Sent SMS history/records

  • The records of all the sent Odoo SMS is maintained in Odoo.
  • The User can view the details of each sent SMS by opening the record.

Auto-Delete the Record

  • Enable Auto-delete Feature for Odoo SMS.
  • Auto-delete can be set at individual template level.
  • If enabled, sent SMS list and delivery status are not stored in Odoo for those SMS.

View Delivery Status of Sent SMS

  • The delivery status of the sent Odoo SMS is available in Odoo.
  • Check the delivery status for SMS as -  sent, failed or delivered.
  • Admin View
  • SMS Form
  • SMS GateWay Configuration
  • Change Log

Create SMS Templates

Create Odoo SMS templates in advance and use them when you send SMS notifications to the customers or vendors.

Choose SMS template for Global Use Or Condition-Based Use.

  • Global

It can be selected in the SMS form to send messages notifications.


  • Condition Based

Used to trigger conditions to automatically send Odoo SMS notifications to the customer when the condition is fulfilled in Odoo.

Enable/Disable Auto-Delete

Enable Auto-Delete so the sent Odoo SMS and their Delivery report is not saved in Odoo.

  • Enable Auto-delete In SMS templates

  • Enable Auto-Delete for each SMS / bulk SMS.

Save Odoo SMS Drafts

Odoo SMS History and Delivery Reports

  • Maintain Sent Odoo SMS Notifications History

  • Delivery reports For Odoo SMS

Create SMS Groups to Send Bulk Odoo SMS Notifications

Create SMS groups include only Customers or vendors, or both.

Use SMS form to send Odoo SMS alerts to customers and/or vendors

Choose recipients as a

  • Group

  • Multiple recipients

  • Single recipient


Select relevant SMS Gateway

Prerequisites

First you need to Install the relevant Odoo SMS Gateway to configure the module.

SMS Gateway Configuration Form

  • Priority setting is for Odoo to determine which SMS Gateway to use while sending SMS.
  • The Odoo SMS gateway with priority set as 1 is used.

Test SMS Gateway Connection

  • Choose the SMS Gateway from the list. (Only installed Gateway shows up on the list)
  • Choose the mobile number to send the test SMS.
